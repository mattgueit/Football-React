# Football-React

## Setup

### Create Typescript / React project with Vite 

1. Run ```npm init vite your-app-name``` or ```npm create vite@latest``` in command line. This will start a prompt.
	1. Select ```React```
	2. Select ```Typescript```
   
2. Run ```cd your-app-name```

3. Run ```npm install```


## Development
### Run development build

```npm run dev```
